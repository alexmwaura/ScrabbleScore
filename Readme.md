# scrabble score
A simple java application for a scrabble word game, where users enter a word and the scrabble score is calculated

## Prerequisites
    1. java 11
    2. Gradle 4.10

## Usage

    1. Clone this repo
    2. open with intellij IDEA
    3. Then run ` javac App.java && java App`


## Testing

   ```java
    gradle test
```
